# Calculator

A feature-rich Python calculator that goes beyond basic arithmetic operations.

## Overview

Calculator is an interactive command-line calculator with memory storage and calculation history tracking. It demonstrates clean Python programming practices including object-oriented design, error handling, and user-friendly input validation.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, and division
- **Advanced Functions**: Power/exponent calculations, square root, and percentage
- **Memory System**: Store, recall, and clear values in memory (M+, MR, MC)
- **Calculation History**: Automatically tracks your last 10 calculations
- **Quick Reuse**: Type 'L' to instantly use your last result in new calculations
- **Error Prevention**: Handles division by zero and invalid inputs gracefully

## How to Run
```bash
python calculator.py
```

## Usage Example

1. Select an operation from the menu (0-13)
2. Enter your numbers when prompted
3. Use 'L' to reuse the last result
4. View your calculation history anytime
5. Store frequently used numbers in memory

## Technical Highlights

- Object-oriented programming with a `SmartCalculator` class
- Input validation and error handling
- Clean, documented code with docstrings
- User-friendly menu-driven interface

Perfect for learning intermediate Python concepts.
